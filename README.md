# Statistical Analysis of NVIDIA Stock Returns

## Objective
To analyze NVIDIA stock returns using descriptive and inferential statistical methods and test whether average daily returns are statistically positive.

## Dataset
Historical NVIDIA (NVDA) stock price data (daily frequency).

## Methodology
- Converted stock prices into daily returns
- Performed inferential statistical analysis
- Conducted one-tailed one-sample t-test

## Key Results
- t-statistic: 2.95
- p-value: < 0.01
- Result indicates statistically significant positive average daily returns

## Tools Used
- Python
- Pandas
- NumPy
- SciPy
